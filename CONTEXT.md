# 地下管网知识模型数据库

本上下文定义 Pipeline-K-RAG 项目讨论中应使用的领域语言。它只记录稳定术语，不记录实现方案、任务计划或架构决策。

## Language

### 产品与范围

**地下管网知识模型数据库**:
面向地下管网、企业文档和空间数据的多用户智能体平台。它把私有知识、知识图谱、空间数据和智能体工具调用纳入同一业务系统。
_Avoid_: 通用聊天机器人、普通知识库系统、单纯 RAG Demo

**Pipeline-K-RAG**:
地下管网知识模型数据库的英文项目名。用于对外材料、仓库描述和跨语言文档时，与“地下管网知识模型数据库”指向同一产品。
_Avoid_: Yuxi、pipeline rag、管网数据库

**Main2.0 能力**:
将知识库、知识图谱、可迁移知识库和空间数据统一接入智能体使用场景的一组产品能力。它不是独立产品线。
_Avoid_: Main2、二期功能、空间图谱模块合集

### 智能体与扩展

**智能体**:
由模型、提示词、工具、知识库、Skills、MCP 和 SubAgents 等配置组成的可运行业务助手。智能体是用户可配置和授权的资源。
_Avoid_: Bot、助手实例、模型配置

**主智能体**:
直接面向用户对话并可调用子智能体完成任务的智能体。它负责承接用户意图和组织工具调用。
_Avoid_: 父 Agent、入口机器人、主 Bot

**子智能体**:
被主智能体调用、用于处理特定任务的智能体。子智能体不再继续配置下一层子智能体。
_Avoid_: 子 Agent、子机器人、嵌套助手

**Skills**:
可挂载到智能体运行过程中的能力包，用于提供特定工作流、工具说明或执行约束。它不是用户业务知识库。
_Avoid_: 插件、脚本集合、知识文件

**MCP 服务器**:
智能体可连接的外部工具能力来源。它用于扩展工具边界，不等同于普通后端 API。
_Avoid_: API 插件、工具接口、远程函数

### 知识库与检索

**知识库**:
带权限边界的私有知识资源集合，可被智能体选择并用于检索。知识库是治理单位，不只是文件夹。
_Avoid_: 文件目录、资料库、数据集

**知识库分组**:
知识库页面中用于归类展示和创建知识库时选择归属的组织层。知识库分组不是权限边界，也不改变知识库作为治理单位的语义。
_Avoid_: 一级文件夹、目录、知识库文件夹

**Milvus 知识库**:
支持文档入库、向量检索、图谱构建、检索评估、迁移和空间数据能力的自建知识库类型。它是本项目的主要知识库形态。
_Avoid_: 向量库、Milvus 表、普通 Collection

**只读连接器**:
连接外部知识源并仅用于检索的知识库类型。它不负责上传、解析、入库或图谱构建。
_Avoid_: 外部知识库、同步源、导入器

**Chunk**:
文档解析后形成的可检索文本片段。Chunk 是检索、图谱抽取和引用溯源的基础知识单元。
_Avoid_: 段落、切片、文本块

**知识图谱**:
从知识库 Chunk 中抽取出的实体、关系及其来源引用。这里的知识图谱属于具体知识库，不是全局共享图谱。
_Avoid_: 全局图谱、关系数据库、Neo4j 页面

**显式图谱检索**:
直接查询并返回实体、关系和子图的检索方式。它强调可见的图结构结果，而不是仅返回文本片段。
_Avoid_: 图谱搜索、关系查询、图数据库查询

**图谱增强检索**:
以图谱实体作为种子扩展关系，再融合图谱命中与 Chunk 命中的检索方式。它用于提升 RAG 对实体关系问题的覆盖。
_Avoid_: GraphRAG、混合搜索、增强搜索

**图谱抽取模型**:
用于从知识库 Chunk 中抽取实体和关系的 Chat 模型配置。它独立于当前对话智能体使用的主模型。
_Avoid_: 对话模型、知识库模型、LLM 抽取器

### 迁移与空间数据

**可迁移知识库包**:
用于跨实例迁移知识库应用层数据的 `.yuxikb.zip` 文件。它包含文档、解析结果、Chunks、图谱数据和必要配置，不包含凭据、用户权限或底层数据库文件。
_Avoid_: 备份包、数据库导出、压缩知识库

**导入预检**:
导入可迁移知识库包前对结构、版本、大小、数量和校验和进行检查的步骤。预检通过不代表已经写入目标知识库。
_Avoid_: 导入校验、上传检查、扫描

**空间数据**:
与地理位置、几何对象和地下管网空间关系有关的数据。它既可以被管理页面展示，也可以被智能体工具查询和分析。
_Avoid_: 地图数据、GIS 文件、图层文件

**空间图层**:
一组可叠加展示和分析的空间要素集合。图层是空间数据的组织单位，不等同于单个文件。
_Avoid_: 图层文件、地图层、GeoJSON 文件

**会话地图**:
智能体对话中返回的交互式地图结果。它用于展示本次会话相关的空间要素和分析结果，不是通用 GIS 管理后台。
_Avoid_: 地图页面、GIS 看板、空间数据管理页

### 治理与运行语义

**多用户资源治理**:
围绕用户、部门、知识库和智能体资源建立的访问控制语义。讨论权限时应优先明确资源归属和授权范围。
_Avoid_: 用户管理、权限系统、账号体系

**运行链路**:
一次智能体请求从用户输入到工具调用、知识检索、事件返回和结果展示的业务路径。它描述产品行为，不表示某个具体函数或目录。
_Avoid_: 调用链、执行流程、请求链路

**轻量模式**:
不启用重依赖知识库能力的运行模式。它只适合有限开发或排障，不代表完整产品验收环境。
_Avoid_: Lite、简化部署、无知识库模式

**隔离验收模式**:
使用独立端口、容器名和数据卷验证生产构建的验收环境。它用于交付前验证，不等同于日常热重载开发环境。
_Avoid_: sync-test、isolated、测试环境
