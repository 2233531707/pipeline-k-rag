# Triage Labels

skills 使用五个标准 triage 角色。本文件把这些角色映射到本仓库 issue tracker 中实际使用的标签字符串。

| mattpocock/skills 中的标签 | 本仓库 tracker 中的标签 | 含义                         |
| -------------------------- | ----------------------- | ---------------------------- |
| `needs-triage`             | `needs-triage`          | 需要维护者评估该 issue       |
| `needs-info`               | `needs-info`            | 等待报告者补充信息           |
| `ready-for-agent`          | `ready-for-agent`       | 规格完整，AFK agent 可接手   |
| `ready-for-human`          | `ready-for-human`       | 需要人类实现                 |
| `wontfix`                  | `wontfix`               | 不会处理                     |

当 skill 提到某个角色时，使用上表中对应的标签字符串。
