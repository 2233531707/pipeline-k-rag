## 用户看不到功能的直接原因

1. `query_knowledge_graph` 是 Agent 中间件工具，不是独立菜单或页面；只有会话启用可访问的 Milvus 知识库，并由模型发起调用时才会出现结果卡片。
2. 知识库迁移前端调用的后端路由不存在，所以导入入口即使显示也不可用；导出入口根本未接到详情页。
3. 图层组合和叠加没有数据库表、API 或前端入口，当前只能查看已导入的单个空间图层。
4. 空间 Agent 工具没有注册，因此会话地图不能从正常 Agent 调用链触发。
5. UI 改版只覆盖了部分页面和 token，未达到指南描述的全产品重设计；隔离环境若仍挂载旧 `web/dist`，还会进一步显示旧页面。
